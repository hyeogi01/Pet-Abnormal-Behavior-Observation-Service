# 🎥 실시간 반려동물 캠 연결 최종 가이드

이 문서는 ngrok 고정 도메인을 사용하여 실제 핸드폰(CCTV)과 노트북(모니터)을 연결하는 최종 단계를 설명합니다.

## 1. 현재 설정 상태 확인
- **서버 주소**: `https://tissue-leverage-diocese.ngrok-free.dev` (이미 `lib/config.dart`에 적용 완료)
- **우회 설정**: ngrok 무료 버전의 경고 페이지를 건너뛰는 헤더 설정 완료.

---

## 2. 테스트 시작 전 실행 순서 (노트북)

### ① ngrok 실행 (가장 먼저!)
터미널을 열고 아래 명령어를 입력하여 터널을 엽니다.
```powershell
.\ngrok http --domain=tissue-leverage-diocese.ngrok-free.dev 8080
```
*(창을 끄지 말고 계속 유지하세요.)*

### ② FastAPI 백엔드 실행
새 터미널 창을 열고 백엔드 폴더에서 서버를 실행합니다.
```bash
uvicorn main:app --host 0.0.0.0 --port 8080
```
*(--host 0.0.0.0 옵션이 있어야 외부 접속을 허용합니다.)*

---

## 3. 앱 빌드 및 설치 (핸드폰용)

### ① APK 파일 생성
`Frontend` 폴더 터미널에서 아래 명령어를 입력합니다.
```bash
flutter build apk --release
```
- 빌드가 완료될 때까지 약 1~3분 정도 소요됩니다.

### ② 앱 전송 및 설치
1. `build\app\outputs\flutter-apk\app-release.apk` 파일을 찾습니다.
2. 이 파일을 **카카오톡(나에게 보내기)**이나 메일 등으로 핸드폰에 전송합니다.
3. 핸드폰에서 파일을 열어 **설치**합니다. (보안 경고 시 '무시하고 설치' 클릭)

---

## 4. 캠 연결 및 모니터링 방법

1. **노트북**: 브라우저에서 모니터링 페이지를 띄웁니다 (`flutter run -d chrome`).
2. **핸드폰**: 설치한 앱을 실행하고 **[공기계로 사용하기]**를 누릅니다.
3. **연결**: 노트북 화면에 떠 있는 **6자리 코드**를 핸드폰에 입력합니다.
4. **성공**: 핸드폰의 카메라 영상이 노트북 브라우저 화면에 실시간으로 나오는지 확인합니다!

---

## ⚠️ 주의사항
- **Wi-Fi**: 핸드폰과 노트북이 같은 Wi-Fi일 필요는 없습니다. (ngrok 덕분에 LTE로도 가능합니다.)
- **백그라운드**: 핸드폰 앱을 끄거나 화면을 잠그면 영상 송출이 중단될 수 있으니 테스트 시에는 화면을 켜두세요.
